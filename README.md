# 基于Rust的后台管理系统

# 功能特点
1. 基于layui和x-admin开发,无缝对接layui前端代码

2. 基于actix-web开发,性能保证,后续无需语言升级

3. MVC 设计模式,快速入门

4. Tera 模板引擎,类django模板引擎,极易上手

5. 基于Rust语言特性,有安全、性能保证
